# Presentación Digital 360° - Cantabria 2 | OTG

Presentación ejecutiva interactiva para la estrategia de marketing digital del proyecto Cantabria 2.

## 📁 Archivos

- `index.html` - Estructura de la presentación
- `styles.css` - Estilos visuales
- `script.js` - Navegación y funcionalidad
- `README.md` - Este archivo

## 🚀 Cómo Usar

### Opción 1: Abrir Localmente
1. Descarga todos los archivos en una carpeta
2. Abre `index.html` en tu navegador (Chrome, Firefox, Safari, Edge)
3. Navega con las flechas del teclado o los botones

### Opción 2: Subir a GitHub Pages
1. Crea un repositorio en GitHub
2. Sube los 3 archivos (index.html, styles.css, script.js)
3. Ve a Settings > Pages
4. Selecciona la rama main y carpeta root
5. Tu presentación estará en: `https://tuusuario.github.io/nombre-repo/`

## 🎯 Navegación

- **Flechas ← →**: Navegar entre slides
- **Espacio**: Siguiente slide
- **Botones**: Click en "Anterior" o "Siguiente"
- **Móvil**: Swipe izquierda/derecha

## 📝 Personalización

Para modificar contenido:
1. Edita `index.html` - Cambia textos, números, tablas
2. Edita `styles.css` - Ajusta colores, tamaños, espaciados
3. No necesitas tocar `script.js` (a menos que cambies cantidad de slides)

## 🔄 Actualizar Cantidad de Slides

Si agregas o quitas slides:
1. Actualiza `const totalSlides = 8;` en `script.js`
2. Asegúrate de que cada slide tenga ID único: `id="slide1"`, `id="slide2"`, etc.

## 💡 Tips

- Presenta en **pantalla completa** (F11 en Windows, Control+Command+F en Mac)
- Usa **Chrome o Firefox** para mejor compatibilidad
